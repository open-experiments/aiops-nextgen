# AIOps Shared Package

Shared components for AIOps NextGen platform.

## Components

- **models/**: Pydantic data models (specs/01-data-models.md)
- **database/**: SQLAlchemy ORM models and session management
- **redis_client/**: Redis client wrapper with pub/sub support
- **config/**: Configuration management with pydantic-settings
- **observability/**: Structured logging and OpenTelemetry setup
